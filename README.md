# SERIAL DATA ACQUISITION

This project is a **Serial Data Acquisition System** that collects data from analog sensors, converts it to digital values using an ADC (Analog-to-Digital Converter), and transmits the collected data via UART for further processing or display. It is capable of switching between multiple input channels to collect data dynamically and features a basic user interface using an LCD.

## Operating Conditions

| Parameter | Specifications |
| :--- | :--- |
| **Storage Temperature** | -40 to 100°C |
| **Input Voltage Range** | 0 to 5V |
| **Output Voltage (UART)** | TTL 5V logic levels |
| **Response Time** | 50 ms per data point |
| **Power Requirement** | 5.0 V DC, 0.1 A input |
| **Temperature Change** | -40 to 85°C |
