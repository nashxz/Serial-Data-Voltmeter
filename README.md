# SERIAL DATA ACQUISITION: MULTI-CHANNEL VOLTMETER

This project is a high-performance **Multi-Channel Voltmeter** built around the **ATMega328p** microcontroller. It utilizes an integrated **LCD circuit** to provide real-time voltage displays while leveraging **USB-to-Serial UART** for seamless data transfer. To ensure precision and responsiveness, the system implements **interrupt-driven real-time data handling**, allowing users to dynamically adjust the sampling frequency via external **push buttons**. This setup enables efficient data acquisition and monitoring across multiple channels with a focus on low-latency processing and user-controlled sampling.
